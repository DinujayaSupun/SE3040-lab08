# SE3040 Lab 08 – React Application

## 📌 Overview
This project is developed for **SE3040 – Application Frameworks Lab 08**.  
It demonstrates basic React concepts such as components, props, and UI structuring.

---

## 🚀 Technologies Used
- React (Vite)
- JavaScript (JSX)
- CSS (Inline Styling)

---

## 📂 Project Features

### ✅ Exercise 01
- Greeting component
- Displays "Hello World!"

### ✅ Exercise 02
- NavBar component
- Navigation menu with links

### ✅ Exercise 03
- Footer component
- Card component (image, title, description)
- Button component (using props)
- Banner component (promotion section)
- Testimonial component (quote, name, image)
